If you have a problem with elements displaying you should try to run Chrome with flags: --allow-file-access-from-files --disable-web-security
To do this, perform the following steps:
1 Close Chrome.
2 Right click on Chrome icon and choose Properties.
3 Add flags (--allow-file-access-from-files --disable-web-security) to the end of target line.
4 Click Apply.

Details on http://www.chromium.org/developers/how-tos/run-chromium-with-flags
